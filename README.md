# QRMaster Pro 🚀
**QRMaster Pro** is a modern, high-performance, offline-first vector QR Code designer and manager. It operates entirely in your client-side browser, ensuring absolute data security, real-time performance, and a sleek, animated glassmorphic interface supporting both Light and Dark modes.

---

### 💎 Features & Customization Engine

### 1. 📂 13 Multi-Protocol Scanners
QRMaster Pro translates form entries directly into standard, stock-compatible QR schemas, meaning they immediately work on standard iOS & Android camera scanners without any custom apps:
*   **Website URLs**: Standard web link directions : https://qrmaster-pro-252335853638.asia-southeast1.run.app
*   **Plain Text**: Multilingual offline text blocks up to 1,200 characters.
*   **WiFi Networks**: Automatic credentials sharing (`WIFI:S:<ssid>;T:<enc>;P:<pass>;;`).
*   **Phone Calls**: Single-tap mobile tel links (`tel:<phone>`).
*   **SMS Vectors**: Pre-filled carrier messaging (`SMSTO:<phone>:<message>`).
*   **Email Mailto**: Prefilled subject line and email body formats.
*   **vCard Contact Files**: Professional contact cards supporting Address, Job Title, Organization, Web URL, Biography notes, and work/cell phones.
*   **Google Maps Location**: Live latitude & longitude queries featuring browser Geolocation retrieve shortcuts.
*   **WhatsApp Deep Links**: One-click instant chat routing (`https://wa.me/...`).
*   **UPI Financial Payments**: Unified Payment interface deep links supporting Payee VPA, display name, amount, billing note, and transactional currencies.
*   **Calendar & Events**: iCalendar-standard `VEVENT` files supporting summary, location, start and end dates.
*   **Social Channels**: Profile link quick managers for Instagram, Twitter/X, LinkedIn, YouTube, TikTok, Facebook, and GitHub.

### 2. 🎨 Advanced Styling & Node Customization
*   **Custom Dot Styles**: Choose between Square, Round Dots, Soft Rounded Capsules, Angle Diamonds, Stars, and Classy Flares.
*   **Finder Pattern Frame Designs**: Customize the 3 corner frames (Square, Rounded, Extra-Rounded, Circular, Leaf and Shield).
*   **Finder Pattern Core Designs**: Customize the inner eye dots (Square, Circle, Diamond, Star, Heart, or Shield).
*   **Color Overrides**: Choose custom colors for dots, outer eye frames, and inner eye cores, featuring transparency toggles.
*   **Centered Brand Logos**: Choose from preloaded vector presets or drag-and-drop custom logos. Control sizing scales, protective masking backs, padding, and corner border clipping.

### 3. 💾 Productivity & Local-First Template Library
*   **Durable Local Database**: Save your configurations as reusable templates in browser `localStorage`.
*   **Filtering & Searching**: Search through saved templates by name, payload content, or categorized platforms.
*   **Starred Favorites**: Fast-track your primary templates with easy star marking.
*   **JSON Import & Export**: Back up, share, or migration-import entire design templates arrays in a single click.
*   **Undo / Redo Managers**: Travel back and forth in configuration edits via dashboard buttons or fast standard shortcuts (`Ctrl + Z` and `Ctrl + Y`).

## 🛠️ Architecture & Technologies

*   **Framework**: React 19 + TypeScript.
*   **Styling**: Tailwind CSS v4.
*   **Bundler**: Vite.
*   **Icons**: Lucide React.
*   **State Animation**: CSS Transition Keyframes.

## ⌨️ Productivity Keyboard Shortcuts

*   **Undo Last Customization**: `Ctrl + Z` (or `Cmd + Z` on Mac)
*   **Redo Last Customization**: `Ctrl + Y` (or `Cmd + Shift + Z` on Mac)
*   **Toggle Expansion Preview**: Hover mouse over preview card and press the expand icon.

## 🔒 Absolute Privacy Guarantee
No servers, no tracking cookies, and no database analytics.
